PhpOpenerp
=

Overview
-
PhpOpenerp is a basic library to access Openerp with PHP.

Usage:
```
$openerp->method('object', $arg1, $arg2);
```
in php is basically the same as
```
self.pool.get('object').method(arg1, arg2)
```
in openerp
